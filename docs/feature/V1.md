# V1: UI/UX Improvements Roadmap

This document tracks the remaining UI/UX improvements for the V1 release of Midnight Invoice.

---

## Tasks

### Invoice List Improvements

- [x] **Clickable invoice list items** - Allow users to click on an invoice row in "Your Invoices" to open the preview
- [x] **Add pointer cursor** - Add `cursor-pointer` styling to invoice list items for better UX
- [x] **Remove preview button** - Remove the dedicated preview button from the invoice list (since clicking the row will preview)

---

### PDF Preview Page

- [x] **Add edit button** - Add an "Edit" button in the PDF preview page to allow quick editing of the invoice

---

### Navigation & Tabs

- [x] **Hide Session History tab** - Remove/hide the Session History tab for now
- [x] **Add Settings tab** - Replace Session History with a Settings tab containing:
  - Default "From" name
  - Default "From" address  
  - Default payment details
  - Default notes
  - **Storage**: Zustand store with persist middleware (localStorage)
  - **UX**: "Use Defaults" button in invoice form to apply saved values
- [x] **Add Coming Soon tab** - Create a new tab to showcase upcoming V2 features
  - **Style**: Simple list with icons and short descriptions
